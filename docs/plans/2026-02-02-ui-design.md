# Taggarr UI Design

A full admin web interface for taggarr, providing monitoring, configuration, and data exploration capabilities.

## Overview

| Aspect            | Decision                                                  |
| ----------------- | --------------------------------------------------------- |
| **Stack**         | FastAPI + TanStack Start/React/TypeScript/Tailwind/shadcn |
| **Deployment**    | Single process serves API + frontend                      |
| **Database**      | SQLite (PostgreSQL optional)                              |
| **Config file**   | YAML, server settings only                                |
| **Auth**          | Single admin user, Forms login, API keys                  |
| **UI role**       | Primary interface, CLI secondary                          |
| **Notifications** | \*arr-style providers (Discord, Telegram, Pushover, etc.) |
| **Backups**       | ZIP archives, scheduled + manual, retention policy        |
| **Base URL**      | Configurable for nginx reverse proxy                      |

```
└─────────────────────┘    └─────────────────────┘
```

**Key points:**

- Single process serves both API and frontend
- SQLite stores all persistent data (migrating from JSON files)
- Background workers handle scheduled scans, notifications, backups
- Configurable via `--port` and `--base-url` flags for nginx compatibility

## Database Schema

Aligned with \*arr patterns: `Id` as INTEGER PRIMARY KEY, key-value Config table, History with EventType enum + flexible Data JSON.

```sql
-- ═══════════════════════════════════════════════════════════════
-- AUTH (matches Sonarr User model)
-- ═══════════════════════════════════════════════════════════════
Users (
    Id              INTEGER PRIMARY KEY,
    Identifier      TEXT NOT NULL,         -- GUID, matches *arr pattern
    Username        TEXT UNIQUE NOT NULL,
    Password        TEXT NOT NULL,         -- hashed
    Salt            TEXT NOT NULL,
    Iterations      INTEGER NOT NULL DEFAULT 10000
)

Sessions (
    Id              INTEGER PRIMARY KEY,
    Identifier      TEXT UNIQUE NOT NULL,  -- session token
    UserId          INTEGER NOT NULL REFERENCES Users(Id),
    ExpiresAt       TEXT NOT NULL          -- ISO timestamp
)

ApiKeys (
    Id              INTEGER PRIMARY KEY,
    Label           TEXT NOT NULL,         -- "Overseerr", "Home Assistant"
    Key             TEXT UNIQUE NOT NULL,  -- hashed
    LastUsedAt      TEXT                   -- ISO timestamp
)

-- ═══════════════════════════════════════════════════════════════
-- CONFIG (matches *arr key-value pattern)
-- ═══════════════════════════════════════════════════════════════
Config (
    Id              INTEGER PRIMARY KEY,
    Key             TEXT UNIQUE NOT NULL,
    Value           TEXT NOT NULL
)

-- ═══════════════════════════════════════════════════════════════
-- INSTANCES
-- ═══════════════════════════════════════════════════════════════
Instances (
    Id              INTEGER PRIMARY KEY,
    Name            TEXT UNIQUE NOT NULL,
    Type            TEXT NOT NULL,         -- "sonarr" | "radarr"
    Url             TEXT NOT NULL,
    ApiKey          TEXT NOT NULL,         -- encrypted
    RootPath        TEXT NOT NULL,
    TargetLanguages TEXT NOT NULL,         -- JSON array ["en", "ja"]
    Tags            TEXT NOT NULL,         -- JSON {"dub": "dub", "semi": "semi-dub", "wrong": "wrong-dub"}
    TargetGenre     TEXT,
    QuickMode       INTEGER DEFAULT 0,
    Enabled         INTEGER DEFAULT 1,
    RequireOriginalDefault  INTEGER DEFAULT 0,
    NotifyOnWrongDub        INTEGER DEFAULT 1,
    NotifyOnOriginalMissing INTEGER DEFAULT 1
)

-- ═══════════════════════════════════════════════════════════════
-- TAGS (matches *arr Tag model)
-- ═══════════════════════════════════════════════════════════════
Tags (
    Id              INTEGER PRIMARY KEY,
    Label           TEXT UNIQUE NOT NULL
)

-- ═══════════════════════════════════════════════════════════════
-- MEDIA
-- ═══════════════════════════════════════════════════════════════
Media (
    Id              INTEGER PRIMARY KEY,
    InstanceId      INTEGER NOT NULL REFERENCES Instances(Id) ON DELETE CASCADE,
    Path            TEXT NOT NULL,
    Title           TEXT NOT NULL,
    CleanTitle      TEXT NOT NULL,
    MediaType       TEXT NOT NULL,         -- "series" | "movie"
    OriginalLanguage TEXT,
    TagId           INTEGER REFERENCES Tags(Id),
    Added           TEXT NOT NULL,
    LastScanned     TEXT,
    LastModified    REAL,
    OverrideRequireOriginal  INTEGER,
    OverrideNotify           INTEGER,
    UNIQUE(InstanceId, Path)
)

Seasons (
    Id              INTEGER PRIMARY KEY,
    MediaId         INTEGER NOT NULL REFERENCES Media(Id) ON DELETE CASCADE,
    SeasonNumber    INTEGER NOT NULL,
    EpisodeCount    INTEGER DEFAULT 0,
    Status          TEXT,
    OriginalDub     TEXT,                  -- JSON array
    Dub             TEXT,                  -- JSON array
    MissingDub      TEXT,                  -- JSON array
    UnexpectedLanguages TEXT,              -- JSON array
    LastModified    REAL,
    UNIQUE(MediaId, SeasonNumber)
)

-- ═══════════════════════════════════════════════════════════════
-- HISTORY
-- ═══════════════════════════════════════════════════════════════
History (
    Id              INTEGER PRIMARY KEY,
    Date            TEXT NOT NULL,
    EventType       TEXT NOT NULL,
    MediaId         INTEGER REFERENCES Media(Id) ON DELETE SET NULL,
    InstanceId      INTEGER REFERENCES Instances(Id) ON DELETE SET NULL,
    Data            TEXT                   -- JSON for flexible metadata
)
-- EventTypes: ScanStarted, ScanCompleted, ScanFailed, TagApplied,
--             TagRemoved, WrongDubDetected, OriginalMissing,
--             InstanceAdded, InstanceUpdated, InstanceRemoved,
--             BackupCreated, BackupRestored, SettingChanged

-- ═══════════════════════════════════════════════════════════════
-- NOTIFICATIONS
-- ═══════════════════════════════════════════════════════════════
Notifications (
    Id              INTEGER PRIMARY KEY,
    Name            TEXT NOT NULL,
    Implementation  TEXT NOT NULL,         -- "Discord", "Telegram", "Webhook", etc.
    Settings        TEXT NOT NULL,         -- JSON
    OnScanCompleted         INTEGER DEFAULT 0,
    OnWrongDubDetected      INTEGER DEFAULT 1,
    OnOriginalMissing       INTEGER DEFAULT 1,
    OnHealthIssue           INTEGER DEFAULT 0,
    OnApplicationUpdate     INTEGER DEFAULT 0,
    IncludeHealthWarnings   INTEGER DEFAULT 0,
    Tags            TEXT                   -- JSON array of instance IDs
)

NotificationStatus (
    Id              INTEGER PRIMARY KEY,
    NotificationId  INTEGER UNIQUE NOT NULL REFERENCES Notifications(Id) ON DELETE CASCADE,
    LastSentAt      TEXT,
    LastError       TEXT,
    ConsecutiveFailures INTEGER DEFAULT 0
)

-- ═══════════════════════════════════════════════════════════════
-- SCHEDULED TASKS
-- ═══════════════════════════════════════════════════════════════
ScheduledTasks (
    Id              INTEGER PRIMARY KEY,
    TypeName        TEXT UNIQUE NOT NULL,
    LastExecution   TEXT,
    LastStartTime   TEXT,
    Interval        INTEGER NOT NULL
)

-- ═══════════════════════════════════════════════════════════════
-- COMMANDS
-- ═══════════════════════════════════════════════════════════════
Commands (
    Id              INTEGER PRIMARY KEY,
    Name            TEXT NOT NULL,
    Body            TEXT,                  -- JSON parameters
    Status          TEXT NOT NULL,         -- "Queued", "Started", "Completed", "Failed"
    QueuedAt        TEXT NOT NULL,
    StartedAt       TEXT,
    EndedAt         TEXT,
    Duration        TEXT,
    Exception       TEXT,
    Trigger         TEXT NOT NULL          -- "Manual", "Scheduled"
)

-- ═══════════════════════════════════════════════════════════════
-- BACKUPS
-- ═══════════════════════════════════════════════════════════════
Backups (
    Id              INTEGER PRIMARY KEY,
    Filename        TEXT NOT NULL,
    Path            TEXT NOT NULL,
    Type            TEXT NOT NULL,         -- "Scheduled" | "Manual"
    SizeBytes       INTEGER,
    CreatedAt       TEXT NOT NULL
)
```

**Auth flow:**

- All `/api/v1/*` routes require auth (session cookie or `X-Api-Key` header)
- Exception: `/api/v1/initialize` only available when no users exist

## UI Pages

```
/                           → Redirects to /dashboard (or /setup if uninitialized)

/settings                   → Tabbed settings page
  /settings/general         → Port, base URL, intervals, log level
  /settings/instances       → Instance management
  /settings/notifications   → Notification channels
  /settings/backup          → Backup settings & manual backup/restore
  /settings/security        → Change password, API keys

/activity                   → Tabbed activity view
  /activity/history         → Event history table
  /activity/queue           → Active/queued commands
  /activity/logs            → Real-time log viewer

/system                     → System info, health, about
```

## Notification System

### Supported Providers

| Provider     | Config Fields                                                   |
| ------------ | --------------------------------------------------------------- |
| **Discord**  | `webhookUrl`                                                    |
| **Telegram** | `botToken`, `chatId`                                            |
| **Slack**    | `webhookUrl`                                                    |
| **Pushover** | `userKey`, `apiToken`, `priority`, `sound`                      |
| **Gotify**   | `serverUrl`, `appToken`, `priority`                             |
| **Ntfy**     | `serverUrl`, `topic`, `priority`, `authToken?`                  |
| **Email**    | `server`, `port`, `ssl`, `username`, `password`, `from`, `to[]` |
| **Webhook**  | `url`, `method`, `headers?`, `body?`                            |
| **Apprise**  | `urls[]`                                                        |

### Event Types

| Event               | Default Message                                                         |
| ------------------- | ----------------------------------------------------------------------- |
| `ScanCompleted`     | "Scan completed for {instance}: {scanned} media, {tagged} tags applied" |
| `WrongDubDetected`  | "Wrong dub: {title} has unexpected language(s): {languages}"            |
| `OriginalMissing`   | "Original missing: {title} doesn't have {original} as default audio"    |
| `HealthIssue`       | "Health issue: {message}"                                               |
| `ApplicationUpdate` | "Taggarr update available: v{version}"                                  |

## Configuration

## Migration Path

### From Current taggarr

**Phase 1: First Run Detection**

- Old YAML with instances? Migrate to database, convert to server-only format
- JSON scan files exist? Import to SQLite
- No existing data? Fresh install, show setup wizard

**Phase 2: Backward Compatibility**

- Old CLI `taggarr scan` still works but deprecated
- JSON files preserved after import
- Backup of old config created before migration
